# LuminateApp
Universal wireless lighting controller for (eventually) all major commercial wireless light providers.

Quick start (run UI):

1. Install dependencies: `pip install -r requirements.txt`
2. Run the web UI: `python web_ui.py`
3. Open http://localhost:5000 in your browser and use the "Turn All On" / "Turn All Off" buttons.

Notes: the app reads keys and hub info from `app_ref.yaml`.
